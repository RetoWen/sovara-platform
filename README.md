# Sovara Platform

Sovara Platform is a cloud-based collaboration and project management solution built for engineering teams.

## Overview

This repository contains the source code for the Sovara Platform, which includes:
- Real-time collaboration engine
- Sprint and kanban management
- Developer tooling and integrations
- Security and compliance modules

## Integration

This repository is connected to the [rready](https://rready.com) innovation platform for development workflow tracking.
